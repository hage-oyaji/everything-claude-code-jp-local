# NanoClaw REPL

`scripts/claw.js` を実行・拡張する際にこのスキルを使用する。

## 機能

- 永続的なMarkdownベースのセッション
- `/model` によるモデル切り替え
- `/load` による動的スキルローディング
- `/branch` によるセッション分岐
- `/search` によるクロスセッション検索
- `/compact` による履歴圧縮
- `/export` によるmd/json/txtへのエクスポート
- `/metrics` によるセッションメトリクス

## 運用ガイダンス

1. セッションはタスクに集中させる。
2. 高リスクな変更の前に分岐する。
3. 重要なマイルストーン後に圧縮する。
4. 共有やアーカイブの前にエクスポートする。

## 拡張ルール

- 外部ランタイム依存関係をゼロに保つ
- Markdownをデータベースとして使用する互換性を維持する
- コマンドハンドラーを決定論的かつローカルに保つ
