# エンタープライズエージェントオペレーション

単一のCLIセッションを超えた運用制御が必要な、クラウドホストまたは継続的に稼働するエージェントシステムに本スキルを使用します。

## 運用ドメイン

1. ランタイムライフサイクル（開始、一時停止、停止、再起動）
2. オブザーバビリティ（ログ、メトリクス、トレース）
3. 安全制御（スコープ、権限、キルスイッチ）
4. 変更管理（ロールアウト、ロールバック、監査）

## ベースライン制御

- イミュータブルなデプロイアーティファクト
- 最小権限のクレデンシャル
- 環境レベルのシークレットインジェクション
- ハードタイムアウトとリトライバジェット
- 高リスクアクションの監査ログ

## 追跡すべきメトリクス

- 成功率
- タスクあたりの平均リトライ回数
- 復旧までの時間
- 成功タスクあたりのコスト
- 障害クラスの分布

## インシデントパターン

障害が急増した場合：
1. 新しいロールアウトを凍結
2. 代表的なトレースをキャプチャ
3. 障害ルートを分離
4. 最小限の安全な変更でパッチ適用
5. リグレッション＋セキュリティチェックを実施
6. 段階的に再開

## デプロイメント統合

本スキルは以下と連携します：
- PM2ワークフロー
- systemdサービス
- コンテナオーケストレーター
- CI/CDゲート
