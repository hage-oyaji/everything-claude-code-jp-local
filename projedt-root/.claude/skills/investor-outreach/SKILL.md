# 投資家アウトリーチ

短く、パーソナライズされ、アクションしやすい投資家向けコミュニケーションを書きます。

## アクティベーション条件

- 投資家へのコールドメール作成
- ウォームイントロリクエストのドラフト
- ミーティング後または無返信後のフォローアップ送信
- プロセス中の投資家アップデートの作成
- ファンドのテーゼやパートナーフィットに基づくアウトリーチのカスタマイズ

## コアルール

1. すべてのアウトバウンドメッセージをパーソナライズする。
2. アスクを低摩擦にする。
3. 形容詞ではなく、証拠を使う。
4. 簡潔に保つ。
5. どの投資家にも送れるようなジェネリックなコピーは決して送らない。

## コールドメールの構造

1. 件名: 短く具体的に
2. オープナー: なぜこの投資家なのか
3. ピッチ: 会社が何をしているか、なぜ今か、どの証拠が重要か
4. アスク: 1つの具体的な次のステップ
5. 署名: 名前、役職、必要に応じて1つの信頼性アンカー

## パーソナライゼーションのソース

以下の1つ以上を参照:
- 関連するポートフォリオカンパニー
- 公開されたテーゼ、トーク、投稿、または記事
- 共通のコネクション
- 投資家のフォーカスとの明確な市場またはプロダクトフィット

そのコンテキストがない場合は、尋ねるか、ドラフトがパーソナライゼーション待ちのテンプレートであることを明記してください。

## フォローアップの頻度

デフォルト:
- 0日目: 初回アウトバウンド
- 4〜5日目: 新しいデータポイント1つを含む短いフォローアップ
- 10〜12日目: クリーンなクロージングを含む最終フォローアップ

ユーザーがより長いシーケンスを望まない限り、それ以上のナッジは行わないでください。

## ウォームイントロリクエスト

コネクターの負担を軽くする:
- イントロがフィットする理由を説明
- 転送可能なブラーブを含める
- 転送可能なブラーブは100語以内に保つ

## ミーティング後のアップデート

含めるもの:
- 議論した具体的な内容
- 約束した回答またはアップデート
- 可能であれば1つの新しい証拠ポイント
- 次のステップ

## クオリティゲート

納品前に:
- メッセージがパーソナライズされている
- アスクが明示的である
- 無駄な言葉や懇願するような言葉がない
- 証拠ポイントが具体的である
- 語数が簡潔に保たれている
