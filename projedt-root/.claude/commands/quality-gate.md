# 品質ゲートコマンド

ファイルまたはプロジェクトスコープに対してECC品質パイプラインをオンデマンドで実行します。

## 使い方

`/quality-gate [path|.] [--fix] [--strict]`

- デフォルトターゲット：カレントディレクトリ（`.`）
- `--fix`：設定されている場合、自動フォーマット/修正を許可
- `--strict`：サポートされている場合、警告でも失敗にする

## パイプライン

1. ターゲットの言語/ツールを検出。
2. フォーマッターチェックを実行。
3. 利用可能な場合、リント/型チェックを実行。
4. 簡潔な修正リストを出力。

## 備考

このコマンドはフックの動作と同等ですが、オペレーターが手動で呼び出します。

## 引数

$ARGUMENTS:
- `[path|.]` オプションのターゲットパス
- `--fix` オプション
- `--strict` オプション
