# PHP パターン

> このファイルは [common/patterns.md](../common/patterns.md) を PHP 固有の内容で拡張します。

## 薄いコントローラ、明示的なサービス

- コントローラはトランスポートに集中: 認証、バリデーション、シリアライゼーション、ステータスコード。
- ビジネスルールは HTTP のブートストラップなしでテストしやすいアプリケーション/ドメインサービスに移動。

## DTO とバリューオブジェクト

- リクエスト、コマンド、外部 API ペイロード用の多形状な連想配列を DTO に置き換え。
- 金額、識別子、日付範囲、その他の制約された概念にはバリューオブジェクトを使用。

## 依存性注入

- フレームワークグローバルではなく、インターフェースまたは限定されたサービスコントラクトに依存。
- サービスロケーターの検索なしでテスト可能にするため、コラボレーターをコンストラクタ経由で渡す。

## 境界

- モデルレイヤーが永続化以上の処理を行っている場合、ORM モデルをドメインの判断から分離。
- サードパーティ SDK を小さなアダプターでラップし、コードベースの残りが自分たちのコントラクトに依存するようにする。
